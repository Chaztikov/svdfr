Main program: svdfr3.m

svdfr3.m  —  Computes the singular value decomposition of a frequency response operator
svdfr3i.m  —  Computes the singular value decomposition of a frequency response operator using the inverse problem
HinfNorm.m  —  Computes the \(\mathcal H_\infty\) norm of a differential operator
all Matlab files  —  Zip file containing all Matlab files

Examples

reactionDiffusion.m  —  Reaction-diffusion equation with Neumann boundary conditions
EulerBernoulli.m  —  Simply supported Euler-Bernoulli beam
schmidSols.m  —  3D channel flow, repeating calculations in Figure 4.10 in Stability and transition in shear flows
visco2D.m  —  2D viscoelasic channel flow
visco2DSingle.m  —  2D viscoelasic channel flow for a single value of frequency
